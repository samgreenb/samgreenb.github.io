PROJECTS

Simulating flocking behavior in a real time performance budget

In this project I developed a real time solution for boids simulation running in the GPU capable of simulating 15,000 simultaneous agents in real time on a consumer grade GPU, a laptop RTX 3060, and over 50,000 in a high end machine with a RTX 4090.

  • • Made in Unreal with C++, HLSL and blueprints.
  • • 3 months of development time.
  • • Made in 2024.

Toy Rendering Engine using DirectX11

To get a working understanding of graphics rendering I worked on building a simple rendering application where I could test both the C++ side of graphics rendering, writing the engine itself, and the shader side using HLSL. I also used RenderDoc to understand better the rendering process and debug the application during development.

  • • Built using the DirectXTK toolkit.
  • • Wrote the engine code, resource management and loading in C++.
  • • Wrote pixel and vertex shaders for the rendering including implementations of shadow mapping, normal mapping and gaussian spot highlights.
